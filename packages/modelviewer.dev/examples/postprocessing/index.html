Toggle Effects

This example demonstrates the use of PostProcessing Effects with the model-viewer. Enabling an effect is as simple as adding the attribute to the model-viewer element, or enabling it using the property. For example, enabling a Bloom Effect which is useful for emissive meshes can be done by adding the bloom-effect attribute, or setting the bloomEffect attribute to true. All implemented effects can be seen in this example, you can toggle them to see the difference. Note: the order of the effect attributes does not matter.
