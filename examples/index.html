<model-viewer> - Examples

There are several HTML pages containing usages of the <model-viewer> element with different configurations for demonstrative purposes and testing specific scenarios.

Demos & Scenarios

  • default.html: The barebones default case.
  • background-color.html: Using the background-color attribute to style the background color.
  • background-image.html: Using the background-image attribute to specify an equirectangular projection image.
  • augmented-reality.html: Tests using configurations that enable viewing the model in augmented reality.
  • loading.html: Examples using the different configurations of displaying an image while loading, and lazy loading models.
  • sources.html: Tests different configurations of loading model formats.
  • multiple-elements.html: Stress test of rendering 16 elements that require rendering on every frame, all in the view at once.
  • list.html: Test of many elements on a page, and lazily rendering only when in view.
  • pbr.html: Showcase of high-resolution models with multiple textures and environment maps.
